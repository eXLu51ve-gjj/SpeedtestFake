# SpeedtestFake 🚀

Приложение для модификации результатов тестирования скорости интернета на **speedtest.net** с помощью Selenium и JavaScript инъекций.

## Возможности ✨

- 🎯 **Кастомизация скорости** — установите любое значение скорости с префиксом
- 📊 **Подмена ping** — установите нужное значение пинга (опционально)
- 🌐 **Фиксация IP-адреса** — введите IP вместо реального (опционально)  
- 📡 **Смена провайдера** — измените отображаемый ISP (опционально)
- 🎨 **Удобный интерфейс** — красивое Tkinter окно с предпросмотром

## Требования 📋

| Требование | Статус |
|-----------|--------|
| **Python** (для исходного кода) | ✅ 3.8+ |
| **Google Chrome** | ⚠️ **ОБЯЗАТЕЛЬНО** |
| **Selenium** | Встроено в exe |
| **Tkinter** | Встроено в exe |

### ⚠️ Важно: Google Chrome

Приложение использует **Selenium WebDriver** для автоматизации браузера. Убедитесь что:
- Google Chrome **установлен на компьютере**
- Chrome находится в стандартной директории (обычно: `C:\Program Files\Google\Chrome\Application\chrome.exe`)

## Установка и использование 🔧

### Вариант 1: Готовый EXE (рекомендуется для пользователей)

```bash
# Просто скачайте и запустите
SpeedtestFake.exe
```

1. Введите значения:
   - **Префикс скорости** (обязательно) — например, `11` → результаты будут `11500 Mbps`, `11800 Mbps`
   - **Ping** — оставьте пусто для реального ping
   - **IP адрес** — оставьте пусто для реального IP
   - **Провайдер** — оставьте пусто для реального провайдера

2. Нажмите **"🚀 Запустить"**
3. Откроется Chrome с speedtest.net
4. Тест автоматически модифицируется в реальном времени

### Вариант 2: Исходный код Python

```bash
# Установка зависимостей
pip install selenium

# Запуск
python SpeedtestFake.py
```

## Сборка EXE 🔨

```bash
pip install pyinstaller selenium

# Сборка
python -m PyInstaller SpeedtestFake.spec --clean

# Результат: dist/SpeedtestFake.exe
```

## Как это работает ⚙️

1. **Tkinter окно** собирает параметры от пользователя
2. **Selenium** запускает Google Chrome с отключенной автоматизацией
3. **JavaScript** встраивается в страницу через `addScriptToEvaluateOnNewDocument`
4. Модификатор **перехватывает**:
   - Canvas отрисовку (спидометр)
   - DOM `textContent` (HTML элементы)
   - Запросы к API (JSON результаты)
5. **Все значения в реальном времени** подменяются на введенные

## Технические детали 🔬

### Модификатор Canvas
- Перехватывает `fillText` и `strokeText`
- Подменяет значения скорости только в зоне спидометра (y >= 120)
- Для ping используется зона y < 120

### Модификатор DOM
- Отслеживает классы элементов (`.download-speed`, `.upload-speed`, `.result-data-large`)
- Избегает двойной модификации (Canvas + HTML)
- Проверяет каждые 250ms для обновления значений

### Финальные результаты
- Проверяет блок с результатами (`.result-data-large.number.result-data-value`)
- Применяет модификацию перед сохранением результатов

## Лимитации ⚠️

- ❌ Только для локального тестирования
- ❌ Не может модифицировать результаты после сохранения на speedtest.net
- ❌ Требует Google Chrome (не поддерживает Firefox, Edge)
- ⚠️ Работает с https://www.speedtest.net (основной сайт)
